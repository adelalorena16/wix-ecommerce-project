# Wix E-commerce Project

This university project consists of an e-commerce website built in Wix for a beauty store called **A&A Beauty**.

## Project Overview
The website includes:
- product categories
- product pages
- shopping cart
- user account area
- wishlist
- newsletter subscription
- automated welcome email
- additional e-commerce features

## Main Features
- Product categories: Lipsticks, Foundations, Eyeshadows
- 11 products with pricing, stock, descriptions, attributes, and variants
- User account area with My Orders and My Wishlist
- Newsletter form with email automation
- Shade Finder page with product recommendations
- Promotions, discount code, best sellers, sale page, gift card, reviews, free delivery, and loyalty points

## Tools Used
- Wix Editor
- Wix Stores
- Wix Members Area
- Wix Reviews
- Wix Forms
- Wix Automations

## Future Improvements
- Dynamic product recommendations for the Shade Finder section using Velo by Wix and JavaScript
